Google ADK - ערכת פיתוח סוכנים

Framework לבניית סוכני AI חכמים

מושגי מפתח

  • Agent - הסוכן הראשי
  • Tool - כלי עבודה
  • @Tool - דקורטור פייתון

למה ADK?

  • לולאת סוכן מובנית
  • כלים מובנים ומסודרים
  • הסקה אוטומטית
  • מודל: gemini-2.0-flash

מבנה קוד ADK

from google.adk import Agent, Tool @Tool def query_erp(entity: str, filters: dict = None) -> dict: """Query Priority ERP API""" url = f"http://localhost:3000/api/{entity}" headers = {"X-API-Key": API_KEY} response = requests.get(url, headers=headers, params=filters) return response.json() agent = Agent( model="gemini-2.0-flash", name="erp_agent", instruction="You are an ERP analysis agent...", tools=[query_erp] )
